1. Core Design Principles (v2)

1.1 Open by default

All knowledge = CC0 or equivalent.
No enclosures.
No proprietary logic.

1.2 Local-first

Nodes retain autonomy.
Federation occurs by choice, not coercion.

1.3 Modular

Every design = a module, improvable by any node.

1.4 Interoperable

Standards for tagging, metadata, file formats.

1.10 Culture-compatible

OSIIN adapts to local cultures; culture does not bow to OSIIN.
